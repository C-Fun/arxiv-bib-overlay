Bookmarklet

Download the chrome extension. After downloading, drag and drop the .crx file into your extensions, which can be found by visiting chrome://extensions in your browser (or clicking Settings → More tools → Extensions).

Extension

Extension

Drag and drop this link to the bookmarks bar in Chrome. Often, the bookmarks bar is hidden which you can toggle by clicking Settings → Bookmarks → Show Bookmarks Bar. For each abstract page visit, click the bookmarklet and the overlay will appear.
