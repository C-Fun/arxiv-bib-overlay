Semantic Scholar ArXiv Overlay

Easy install

Chrome

Please visit the extension's page at the Chrome webstore. From there, follow instructions to add to Chrome.

Firefox

Download the signed extension here. Typically, opening this file will install it in Firefox. If that does not occur automatically, visit Settings → Add-ons, click the gear, Install Add-on From File... and select the downloaded file.


Manual installation

To test the extension please choose either the extension or bookmarklet method. The extension is available for Chrome and Firefox and automatically loads the overlay for CS abstract pages. The bookmarklet works for any browser but requires you to select the bookmarklet for each new abstract page which you visit.

Extension

Chrome

Download the Chrome extension here. After downloading, drag and drop the .crx file into your extensions, which can be found by visiting chrome://extensions in your browser (or clicking Settings → More tools → Extensions). When visiting arXiv abstract pages, the overlay should then be active.

Firefox

Download the a zipped archive of the extension from here. Unzip this archive anywhere on your computer (in the Downloads folder is fine). In Firefox, visit about:debugging and select the button on the far right labelled Load Temporary Add-on. Navigate to the unzipped archive and select the manifest.json file. The extension should then be active.

Bookmarklet

Drag and drop this link to your bookmarks (or bookmarks bar for easy access. In Chrome and Firefox, the bookmarks bar is hidden by default and can be shown by a setting in bookmarks settings). For each abstract page visit, click the bookmarklet and the overlay will appear.
